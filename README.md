# Titanic Survival Analysis (Exploratory Data Analysis)

## Project Overview
This project performs exploratory data analysis (EDA) on the Titanic dataset to understand the factors that influenced passenger survival. By analyzing various attributes such as age, gender, ticket class, and fare, the goal is to uncover patterns that explain survival rates.

Exploratory Data Analysis is an important step in data science because it helps identify relationships between variables, detect missing data, and gain insights before applying machine learning models.

---

## Objectives
- Explore the Titanic dataset structure
- Identify missing values and data inconsistencies
- Analyze survival patterns
- Visualize relationships between survival and passenger features
- Extract meaningful insights from the dataset

---

## Dataset
The dataset used in this project is the **Titanic dataset from Kaggle**, which contains information about passengers aboard the Titanic.

Key columns include:
- PassengerId
- Pclass (Passenger Class)
- Name
- Sex
- Age
- Fare
- Embarked
- Survived

---

## Tools and Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

---

## Project Workflow
1. Import necessary Python libraries
2. Load the dataset
3. Inspect dataset structure and missing values
4. Perform data cleaning and preprocessing
5. Conduct exploratory analysis
6. Create visualizations to identify patterns
7. Interpret results and extract insights

---

## Visualizations Created
- Survival distribution chart
- Survival by gender
- Survival by passenger class
- Age distribution of passengers
- Relationship between fare and survival

---

## Key Insights
- Female passengers had significantly higher survival rates compared to male passengers.
- First-class passengers were more likely to survive than passengers in second or third class.
- Younger passengers showed slightly better survival rates than older passengers.
- Passengers who paid higher fares generally had higher survival probabilities.

---

## Conclusion
The exploratory analysis highlights that gender and passenger class were the most influential factors affecting survival during the Titanic disaster. These insights can later be used to build predictive models for survival prediction.
